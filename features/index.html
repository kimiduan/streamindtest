慧流，简单的智能数据操作引擎

慧流（Streamind IDOL：Intelligent Data Operating Layer）致力于从自然内容中提取出信息元素并赋予算法，通过易于使用的SaaS API的形式输出智能数据。

全球信息量每年的最低增⻓率为59%，其中有15%是结构化数据，其余的85%则由各种非结构化数据组成 —— by Garter，2012

非结构化数据中包括了大量的文档、电子邮件、网页、博客等形式自然内容，这些自然内容通常不会直接包含在数据库中。由于很多用户缺乏能够理解并有效利用这些内容的技术（企业不可能聘请专人审查所有这些内容），这些充满战略意义的丰富内容资源常常无法发挥作用，所以用户对于如何处理数量庞大的非结构化数据有着迫切的需求。

我们提出了一个具有创新性的理论框架——基于人工神经网络的模式识别，这是由香农信息论、贝叶斯概率理论和BP神经网络模型这三种经典理论的独特结合与应用，同时我们利用一些尖端的技术，包括：机器学习，计算机视觉，超大规模的Web采集和分布式计算等，从而能有效的对自然内容进行理解、处理和分析。

慧流引擎从数据源的自然内容中提取出有意义的信息元素，并且让引擎“学习”和“理解”这些信息元素，然后赋予这些信息元素相关算法，再将信息元素按用户需求进行分发，最终实现自然内容到智能数据的转化过程。

智能数据可以实现：

  • 内容管理
  • 自动分类
  • 添加标签
  • 情感分析
  • 关系抽取
  • 文字概念标记
  • 自然语言识别
  • 模糊搜索优化
  • 相关内容推荐
  • 更多

我们认为这是一个从“搜索（Search）”到“发现（Discover）”的过程转变，这将会更有效的帮助用户对于内容进行管理，使企业能自动地理解客户与员工感兴趣的东西、他们的行为以及与各种类型信息之间的交流，能更好的利用员工的知识、经验和专业技能进行协作，帮助用户做出更有效的决策、获得更多的价值。

慧流｜Streamind IDOL API

API文档：点击下载

建议调用方法：cURL

URL：http://www.streamind.com/api/idol

支持格式：JSON

HTTP请求方法：POST

访问受权限制：普通接口，无频次限制

请求参数 必选 类型&范围 说明
access_token true string 开发者验证码 API Key
input_url false string 输入需要处理内容的链接
input_text false text 输入需要处理内容的文本
id_number false int64 输入已处理内容ID号返回历史结果

“author”:“百度百科”,

返回值 字段类型 说明
id_number int64 已处理内容ID号
content text 内容文本
title string 标题
time time 处理时间
author string 作者
source string 来源
